Are Ya Winning Son?

A 3-day project for Game Jam Aotearoa 2022. Developed in Unity.

The theme was "transition", so we made a game about the transition from childhood to adulthood as you gain more responsibilities, and perhaps face a few less-than-desirable physical changes.

Itch.io (playable in browser) https://shonnyy.itch.io/are-ya-winning-son
