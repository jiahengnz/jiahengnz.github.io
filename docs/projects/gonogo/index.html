Go/No-go

A 3-day project at a VR workshop. The game is based on the Go/no-go psychology test that requires the participant to press a button when the screen is green, and not press the button when the screen is red. The experiment is recreated in virtual reality with a spaceship theme. Development was done in Unity for Oculus Quest headsets.
